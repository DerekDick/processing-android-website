Ketai

http://ketai.org/ The Ketai library is designed for working with Android sensors and devices in Processing. It includes classes and examples for using device cameras, hardware sensors, multi-touch gestures, databases, and peer-to-peer networking via Bluetooth, WiFi, WiFi Direct, and NFC.

Cardboard

https://github.com/codeanticode/processing-cardboard This library includes a stereo renderer to create Cardboard apps

SelectFile

http://andrusiv.com/android-select-file/ Android library which provides Dialogs for selectInput(), selectFolder() and selectOutput() methods.

Video for Android

https://github.com/omerjerk/processing-video-android This library tries to expose the same API endpoints as that of processing-video library for PC

Cassette

https://github.com/shlomihod/cassette Processing Audio implementation for Android
