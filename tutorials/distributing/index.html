Distributing apps

In this tutorial we will cover the steps to publish your Processing sketch as an app in the Google Play store.

Creating a keystore

The Android Mode simplifies the publishing of your sketch by signing and aligning the app, so you can upload it to the Google Play Developer Console without any extra additional steps. All you need to do is to select the "Export Signed Package" option under the File menu:

Once selecting this option, Processing will ask to create a new keystore to hold the private release key to sign the app package:

Remember this password, as you will have to use it every time you export a new signed package. However, you can reset it and create a new key.

The signed (and aligned) package will be saved in the bin subfolder the sketch's folder, under the name [Sketch name]-release-signed-aligned.apk. With this file , you can follow the instructions from Google to complete the publishing process.

Setting custom icons

Android mode uses a set of default icons for the app. You should use your own icons by creating the files icon-32, 48, 72, 96, 144, and 192 in .PNG format and placing them in the sketch's folder before exporting the signed package. Follow the icon design guidelines from Google in order to create your icons.
