Distributing apps

How to distribute your sketches as apps that other users can install

Creating a keystore

Processing simplifies the packaging of your sketch as a signed app that can be uploaded to the market, with the "Export Signed Package" option in the mode:

Once selecting this option, Processing will ask to create a new keystore to hold the private key to sign the app:

Remember this password, as you will have to use it when uploading the app package to Google Play, and every time exporting a new sketch, although you can reset it.

The signed package will be saved in a bin folder the sketch's folder, under the name [Sketch name]-release-signed.apk. With this file and the private key, you can follow the instructions from Google.

Setting custom icons

Android mode uses a set of default icons for the app. You can use your own icons by creating the files icon-32, 48, 72, and 96 in png format and placing them in the sketch's folder. Follow the icon design guidelines from Google.
