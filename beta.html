Beta release of the Android mode

The latest stable release of the mode is 3.0, which you can use to create regular Android apps.

Version 4.0 is currently in beta stage and adds support for live wallpapers, watch faces, and Google Cardboard.

The 4.0 pre-releases need to be installed manually by downloading the package from the GitHub releases, check the install page for more details.
