Almost all of the Processing API can be used in the Android mode, with a few exceptions. There is a small number of new functions and constants that are unique to the Android mode, specially relevant to wallpapers, watch faces, and Cardboard apps. However, these will likely change in the next beta releases of the mode, as the API gets cleaned up and refined.
